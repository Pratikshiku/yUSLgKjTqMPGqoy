## College Student Club Activity Platform System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3cbb589c7e244ad9b3ecf6a144a7c850/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users (students), club presidents, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Club Information, Collection, Join, Comments
5. Club Activities, Registration
6. Club Forum, Post Topics, Comment on Posted Topics
7. Club News
8. Personal Center
9. My Posts
10. My Collection
11. Change Password
12. My Club Applications
13. My Activity Registrations


**Club President Functions:**
1. Login
2. Change Password
3. My Information
4. Club Information Management
5. Club Application Management
6. Club Activity Management
7. Activity Registration Management

**Administrator Functions:**
1. Login
2. Personal Center
3. Student Management
4. Club President Management
5. Club Type Management
6. Club Information Management
7. Club Application Management
8. Club Activity Management
9. Activity Registration Management
10. Club Forum
11. Club News Management
12. System Carousel Image Management
